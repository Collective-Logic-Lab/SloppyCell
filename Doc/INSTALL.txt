Installing SloppyCell
=====================

Dependencies
~~~~~~~~~~~~

Required to run
+++++++++++++++
* Python_
* Scipy_
* matplotlib_

Required to build
+++++++++++++++++
* Fortran77 compiler

Recommended
+++++++++++
* LibSBML_: Required for SBML import and export. Be sure to build ``--with-python``.
* Psyco_: A just-in-time compiler for python, Psyco will speed up integrations by about 50% on supported platforms.
* Graphviz_: SloppyCell can generate ``.dot`` files specifying network diagrams,which can be visualized using Graphviz. 
* IPython_: A dramatic improvement over the standard python shell.


Instructions
~~~~~~~~~~~~
.. topic:: On Windows

   1. Install required dependencies. We recommend installing the 
      `Enthought Python distribution <http://www.enthought.com/python/>`_ to 
      satisfy the Python, SciPy, and compiler dependencies. Here is a version of
      `libsbml built against Python 2.3 <libsbml_for_python23_win.zip>`_.
   2. To build SloppyCell on Windows using the compiler included in the 
      Enthought distribution.

      a. Unpack the tarball. (``tar xzf SloppyCell-0.3.tar.gz``)
      b. Descend into the SloppyCell-0.3 directory. (``cd SloppyCell-0.3``)
      c. Build SloppyCell. (``python setup.py build -cmingw32``) 
      d. Run the installation script. (``python setup.py install``) 
      e. Run the included test routines. (``cd Testing; python test.py``)

   3. Or just run the executable installer from our `download page <http://sourceforge.net/project/showfiles.php?group_id=140498>`_.

.. topic:: On OS X and Linux

   1. Install required dependencies.  
      (On OS X 10.3, we recommend installing 
      `MacEnthon <http://download.enthought.com/MacEnthon/>`_ which will install
      Scipy, matplotlib, and IPython. A pre-built version of g77 (Fortran77 
      compiler) for OSX 10.3 is available at http://hpc.sf.net.)
   2. Unpack the tarball. (``tar xzf SloppyCell-0.3.tar.gz``)
   3. Descend into the SloppyCell-0.3 directory. (``cd SloppyCell-0.3``)
   4. Run the installation script. (``python setup.py install``) 
   5. Run the included test routines. (``cd Testing; python test.py``)
